## Metasploit
---
> Metasploit is a powerful penetration testing framework and is specially used to find,explore,validate and verify vulnerabilities in systems. It allows researches and ethical hackers to verify if a given system has any vulnerabilities and automate attacks, create payloads, and manage sessions after explanations.
> It has quite a few useful tools such as:
> - msfconsole => main command-line interface
> - msfvenom => mainly used to generate payloads
> - meterpreter => advanced,stealthy payload for post-exploitation

## Burp Suite
---
> Integrated platform for web application security testing.
Commonly used to intercept, modify, and analyse HTTP/S requests.

It includes the following Key Components:
> Proxy => Intercepts and modifies web traffic between browser and target
> Repeater => Manually resends and tweaks HTTP requests
> Intruder => Automate attacks such as fuzzing or brute force
> Scanner(Pro) => detects common web vulnerabilities like XSS and SQLi

## Escalação de Privilégios
---
Process of gaining higher access permissions in a system than initially granted
>Vertical Escalation => from normal user to root/admin
>Horizontal Escalation => gaining access to another user's privilege
>Commonly used techniques:
1. Exploit Kernel or service vulnerabilities
2. Misconfigured sudo permissions
3. Reading passwords in plaintext files
4. abusing scheduled tasks(cron jobs)
